Google Playstore Reviews Listing API

This is a simple script designed to pull reviews from the Google Play App Store for any app using the App ID parameter.

Visit the playstore, access an app's details page and copy the app ID from the URL. eg: https://play.google.com/store/apps/details?id=com.supercell.clashofclans&hl=en

Enter the App ID of any app and hit the 'Get Reviews' button to see the results.
